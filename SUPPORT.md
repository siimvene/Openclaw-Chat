# Support for OpenClaw Chat

## Getting Help

### GitHub Issues

For bug reports, feature requests, and general questions, please use [GitHub Issues](https://github.com/siimvene/Openclaw-Chat/issues).

When reporting a bug, please include:
- iOS/iPadOS version
- Device model
- Steps to reproduce the issue
- Expected behavior
- Actual behavior
- Any error messages

### Common Issues

#### Cannot Connect to Gateway

1. Verify your gateway URL is correct
2. Ensure your gateway is running and accessible
3. Check that your access token is valid
4. If using Tailscale, ensure your device is connected to the tailnet

#### Voice Mode Not Working

1. Grant microphone permissions when prompted
2. Grant speech recognition permissions when prompted
3. Check that your device isn't in Silent mode for TTS output

#### Messages Not Sending

1. Check network connectivity
2. Verify gateway is online (check Status tab)
3. Try disconnecting and reconnecting

## OpenClaw Gateway Issues

For issues with the OpenClaw gateway itself (not this iOS app), please refer to the [OpenClaw documentation](https://openclaw.io).
