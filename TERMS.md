## Description of Service

OpenClaw Chat is a mobile client application that connects to OpenClaw gateway instances. The App enables you to:
- Send and receive messages through your OpenClaw gateway
- Use voice input and text-to-speech features
- Manage chat sessions
- Monitor gateway status and usage
